function knownToolNames(baseTools: readonly ToolDefinition[]): KnownToolNames

The static name universe for profile validation: each runtime-owned tool family's exported constant plus every base definition's name, split by terminality. A base name that collides with a family tool (or another base tool) would silently shadow it at selection time, so collisions are a startup error like every other config fault.
