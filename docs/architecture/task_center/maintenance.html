This support page records how the TaskCenter wiki was built and how future agents should refresh it. It is documentation metadata only; runtime code is not owned here.

Refresh Triggers #

Workflow Iteration Attempt

Refresh when Workflow, Iteration, Attempt, nested planning depth rules, retry, deferral, delegated-workflow routing, close-report delivery, or store protocol methods change.

Attempt Harness

Refresh when planner submission schemas, generator DAG semantics, evaluator launch, launch failure, exhaustion, or audit event behavior changes.

Context Engine

Refresh when recipes, packet block kinds, renderer compression, tag dictionary, what-in-context output, task guidance, skills, or launch message row rules change.

Agent Roles

Refresh when profile frontmatter, allowed tools, terminal sets, notification triggers, main role contract, helper profiles, or validation changes.

Terminal Tools

Refresh when submission tool names, schemas, terminal descriptors, advisor review focus, output validation, or terminal exclusivity changes.

Cross-Module Bridges

Refresh when TaskCenter entry bootstrap, sandbox binding, run_ephemeral_agent, ExecutionMetadata, message artifacts, or sandbox API routing changes.

Stale Claims #

  • docs/task_center_harness_and_context_engine.html remains historical reference material, not the canonical current wiki. Use this module for refreshed source-backed TaskCenter claims.
  • Current state-service naming is WorkflowLifecycle plus IterationAttemptCoordinator, not GoalHandler, IterationManager, IterationFactory, or IterationClosureRouter.
  • Current child-goal closure completes the parent generator task as DONE or FAILED. Do not document it as returning the parent to RUNNING.
  • Current nested-depth terminal routing can remove submit_execution_handoff from nested executors. Do not repeat older claims that executor handoff is available at every nesting depth.
  • The current helper agent uses direct launch prompt assembly through ask_advisor; the Context Engine is not involved in helper prompt composition.
